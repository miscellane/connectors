Tableau Web Data Connectors

The Tableau Web Data Connector enables connection to a web data source, particularly JSON data files. The web data connectors developed herein depend on GitHub JSON files.

The Tableau graphs example below relies on the data of 2 web data connectors
  • capita connector
  • gazetteer connector
The connectors read GitHub JSON files created by greyhypotheses' COVID Tracking Project calculations/summaries. In brief capita connector reads capita.json, whilst gazetteer connector reads gazetter.json. The data sets are joined, via a common key, prior to graph drawing.



Limitations of Tableau Web Data Connectors

Alas, the tools required for automatically updating Tableau graphs based on Tableau web data connector feeds are only available via Tableau Online or Tableau Server; Tableau Public projects can't be updated automatically. Hence, I will not be using Tableau Web Data Connectors presently.
